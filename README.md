# Book Finder

[![License: MIT](https://img.shields.io/badge/License-MIT-brightgreen.svg)](https://opensource.org/licenses/MIT)

## Description

This is a simple book search engine. You can also add the book to your favorite list by register an account.

- [Usage](#usage)
- [License](#license)

## Installation

To install necessary dependencies, run the following command:

```bash
npm i
```

## Usage

To start React app and the server, run:

```bash
npm run develop
```
